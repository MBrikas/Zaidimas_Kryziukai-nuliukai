# Zaidimas_Kryziukai-nuliukai
## Žaidimas Kryžiukai-nuliukai. Python kursai Code Academy PT13

Kryžiukų-nuliukų žaidimas dviems žaidėjams.

Žaidimo paleidimui naudoti -  main_menu.py. 
main_menu.py savyje turi trumpai aprašytas žaidimo taisykles rodomas naudotojui. Žaidimo paleidimo mygtuką. Taip pat po pasibaigusios  žaidimo partijos atlieka žaidimo nunulinimo funkciją, ištrina tarpinius žaidimui reikalingus duomenis.

<img width="839" alt="image" src="https://user-images.githubusercontent.com/100312015/156000719-2964f7d5-e305-4124-a1ca-d0985f544911.png">

Paspaudus "Pradėti naują žaidimą ->" aktyvuojamas zaidimas.py ir atsidaro pagrindinis žaidimo langas.

<img width="651" alt="image" src="https://user-images.githubusercontent.com/100312015/156001159-093d07f4-76d9-4599-82a8-5effdfdfbf20.png">

Pelyte paspaudus ant žaidimo lentos langelio atitinkamoje vietoje atsirandą žymeklis. Melynas žaidėjas pradeda žaidimą, tada seka Baltojo eilę ir t.t. 

<img width="643" alt="image" src="https://user-images.githubusercontent.com/100312015/156002309-0eaf61b5-9d43-49db-a789-fb978320e6e6.png">

Žaidėjo tikslas – surinkti tris vienodus ženklus vienoje linijoje (horizontaliai, vertikaliai arba įstrižai).


Galimos 3 žaidimo pabaigos. 

* Mėlynasis žaidėlas laimi (surenka tris vienodus ženklus vienoje linijoje).
* Baltasis žaidėjas laimi (surenka tris vienodus ženklus vienoje linijoje).
* Lygiosios (visi 9 langeliai užimti ir nėra laimėtojo). 

<img width="874" alt="image" src="https://user-images.githubusercontent.com/100312015/156004708-82489f49-c013-43ef-9c53-ca432265cf97.png">



### Dependencies

* Python version with Tk support.

### Executing program

* Run main_menu.py
* Read main rules of the game and click "Pradėti naują žaidimą ->" button to start the game.
* Use mouse clicks to place the game pieces.
* Popup window automatically informs about the match end. Click "OK" and additional click on the game canvas to close the game window.
* For another match click "Pradėti naują žaidimą ->" button.
